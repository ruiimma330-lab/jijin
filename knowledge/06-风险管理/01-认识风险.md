# 认识风险

## 一句话理解

> 投资风险不是"可能亏钱"——**投资风险是你不知道自己在干什么。**

知道自己为什么买、能亏多少、亏了怎么办——这叫风险管理。不知道——那叫赌博。

## 风险不是坏东西

大多数人的第一反应：风险 = 坏 → 我要避开。

**但投资里，风险和收益是一枚硬币的两面：**

```
银行存款：风险 ≈ 0，收益 ≈ 1.5%  → 跑不赢通胀
债券基金：风险低，  收益 3-5%    → 勉强保值
股票基金：风险高，  收益 8-12%   → 能增值
```

**你永远不可能同时享受股票的高收益和存款的低风险。** 这是一个必须接受的权衡。

关键是找到**你能承受的风险水平**，然后在这个水平上获取最好的收益。

## 投资里你会遇到的四种风险

### 一、市场风险（系统性风险）

```
整个市场一起跌。不管买什么股票都在跌。

原因：经济衰退、战争、疫情、政策变化...

能否通过分散来消除：❌ 不能
怎么应对：
  - 降低股票仓位（多配债券/现金）
  - 拉长持有时间（历史表明市场最终会回来）
  - 定投（越跌越买，摊低成本）
```

### 二、个体风险（非系统性风险）

```
某一家公司出事，但其他公司没事。

例：某白酒公司被查出质量问题，股价崩了
   但白酒行业其他公司没影响

能否通过分散来消除：✅ 可以
怎么应对：
  - 买基金而不是个股（一篮子股票天然分散）
  - 宽基指数基金包含了上百家公司
```

> **买指数基金最大的好处：消除个体风险。**

### 三、流动性风险

```
想卖的时候卖不掉，或者必须以很低的价格卖。

房产：挂牌半年没人买 → 流动性很差
A 股基金：T+1 赎回，2-3 天到账 → 流动性很好
某些理财产品：锁定 1 年不能赎回 → 流动性差

怎么应对：
  - 留够活期/货基做应急金
  - 长期投资的钱才放进锁定期产品
```

### 四、通胀风险

```
钱放着不动，购买力悄悄缩水。

100 元今年能买 5 斤猪肉。
通胀 3%，明年只能买 4.85 斤了。
→ 什么也不买也是一种风险！

怎么应对：
  - 至少把大部分钱放进跑赢通胀的资产（股票/债券）
  - 长期来看，现金是确定在贬值的
```

## 怎么判断自己能承受多大的风险

### 问题一：这笔钱多久后要用？

```
< 1 年  →  货币基金 / 银行存款   →  别碰股票
1-3 年  →  债券为主 + 少量股票  →  可以配 20% 股票
3-5 年  →  股债各半              →  可以配 50% 股票
> 5 年  →  股票为主 + 债券       →  可以配 70%+ 股票
```

**时间越长，你能承受的短期波动越大。** 因为你有时间等市场恢复。

### 问题二：如果亏了 X%，你会怎么做？

诚实地问自己：

```
假设投了 10 万：

跌 10%（剩 9 万）    →  A. 无所谓  B. 有点慌  C. 想卖
跌 20%（剩 8 万）    →  A. 加仓    B. 扛着    C. 割肉
跌 30%（剩 7 万）    →  A. 大举加仓 B. 关账户不看 C. 这辈子不碰

如果你大部分选 C → 你是保守型 → 多配债券
如果你大部分选 A → 你是进取型 → 但也要控制仓位
```

> **在平静的时候想好最坏情况，比在恐慌时做决定好 100 倍。**

### 问题三：你的"睡眠线"在哪里？

```
每个人的风险承受力不同：

小张：账户亏 5% 就失眠 → 多配债券
小李：亏 20% 没感觉 → 可以多配股票
小王：亏 30% 还开心"打折了！"→ 可以更多股票

你的睡眠线是多少？
```

## 控制风险的四个工具

### 工具一：仓位管理

```
永远不满仓，永远不空仓。

100% 股票 → 市场跌 50%，你没钱补仓
0% 股票   → 市场涨 100%，你踏空

留 10-20% 的现金/债券 → 市场崩盘时的弹药 + 心理安慰
```

### 工具二：止损（但要慎用）

```
基金一般不设硬止损（割肉往往割在最痛的时候）
但可以设"软止损"：

"如果基金经理换了，我考虑卖"
"如果基金规模暴涨到 200 亿，我考虑卖"
"如果我当初买它的理由不成立了，我卖"

不要因为"跌太多了"而卖——那通常是最差的卖出理由。
```

### 工具三：分散（最重要的免费午餐）

```
股票 + 债券 + 海外 + （黄金/商品）= 

它们一般不会同时跌：
  股票大跌时，债券往往在涨
  A 股不好的年份，海外可能好

→ 组合整体的波动远小于任何单一资产
→ 而且长期收益没有明显降低

这是投资里唯一一个真正的"免费午餐"。
```

### 工具四：时间（最被低估的工具）

```
持有 1 年：赚钱概率约 60-70%
持有 3 年：赚钱概率约 80-85%
持有 5 年：赚钱概率约 90-95%
持有 10 年：赚钱概率接近 100%（对宽基指数而言）

时间越长，短期波动越不重要。
你最大的风险不是"短期跌了多少"，
而是"没有给它足够的时间长回来"。
```

## 投资前你应该签的"心理合约"

拿出一张纸，写下这些问题的答案。放在你的投资账户旁边。

```
1. 这笔钱是________（金额），我计划________年后才用。
2. 我能接受的最大亏损是________%（具体数字）。
3. 如果跌到那个数，我会________（加仓/不动/减仓），
   因为我不想在恐慌中做决定。
4. 我选择这只基金的原因是________。
   只有在这个原因不再成立时，我才会卖出。
5. 我每月只看账户________次（建议 ≤ 1 次），
   不看就不焦虑。
```

## 这篇你该带走的三句话

1. **风险不是坏东西——但你得认识它、控制它**
2. **时间是你最大的朋友**——持有越久，风险越低
3. **平静时写好规则，恐慌时只执行规则**
