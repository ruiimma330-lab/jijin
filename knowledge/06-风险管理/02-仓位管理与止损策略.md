# 仓位管理与止损策略

## 一句话理解

> 仓位管理就是**控制每次下注的大小**——你不会在一手烂牌时把所有筹码推出去。

## 为什么仓位管理是"最重要但最无聊"的技能

仓位管理不性感。它不像"精准抄底"那样让人兴奋。

但它决定了你能否"活到"下一轮牛市。

### 用一个惨痛的教训说明

```
小王在 2021 年初看到基金涨得好，把全部积蓄 30 万都投了进去。

2021 年春节后 → 跌了 15%
小王想："机会来了！" → 又借了 10 万补仓

2022 年 → 又跌了 20%
小王慌了 → 但已经没钱补了 → 全仓套牢

2022 年 10 月 → 市场最底部
小王忍痛割肉："再也不碰基金了" → 亏了 10 万离场

如果他只投了 50%：
  跌了能补，涨了也不踏空
  心态完全不同
```

## 仓位管理的基本原则

### 原则一：永远不要 All-in

```
因为：
  你不知道市场还能跌多深
  你不知道下跌会持续多久
  你不知道自己能不能扛到最后

留有余地 = 给自己买了一份"选择权"
```

### 原则二：越跌越有价值时，越应该加仓

```
这不是"抄底"，而是：

  1.0 元买 → 觉得合理
  0.8 元买 → 更便宜，应该买更多
  0.6 元买 → 太便宜了，加大力度

前提：你买的是宽基指数，不是可能死掉的个股。
```

### 原则三：对小白来说，定投自动解决了仓位管理

```
每月投入固定金额 →
  → 市场高时，买得少
  → 市场低时，买得多
  → 自动实现了"低位高仓位，高位低仓位"
  
  你完全不需要判断"现在该买多少"，定投帮你做了。
```

## 三种仓位策略

### 策略一：恒定比例（推荐新手）

```
设定一个比例，始终保持。

例：60% 股票 + 40% 债券

当股票涨了 → 股票变多 → 卖股票买债券，恢复 60/40
当股票跌了 → 股票变少 → 卖债券买股票，恢复 60/40

为什么有效：
  → 自动在股票贵时减仓，便宜时加仓
  → 不需要判断市场，只需要执行纪律
```

### 策略二：动态调整（进阶）

```
根据估值调整股票仓位：

PE 分位 < 20%  →  股票 80%  （低估，重仓）
PE 分位 20-40% →  股票 70%
PE 分位 40-60% →  股票 60%  （合理，标配）
PE 分位 60-80% →  股票 45%
PE 分位 > 80%  →  股票 30%  （高估，轻仓）

需要经验判断，新手可以用定投替代。
```

### 策略三：金字塔加减（更进阶）

```
加仓时：越跌买越多
  跌 5%  →  +1000 元
  跌 10% →  +2000 元
  跌 15% →  +4000 元

减仓时：越涨卖越多
  涨 20% →  卖 1/4
  涨 35% →  卖 1/3
  涨 50% →  卖 1/2

但这需要预留大量现金，不适合资金量小的新手。
```

## 止损——基金投资要不要设止损？

### 标准答案是：基金一般不需要硬止损

原因很简单：

```
止损是什么：跌到 X% 就卖

问题：
  1. 基金波动本就很大，跌 15-20% 很正常
  2. 止损后钱放哪？存银行跑不赢通胀
  3. 什么时候买回来？等涨回来再买 = 高买低卖
  4. 反复止损 = 反复割肉 = 累积亏损
```

**对宽基指数基金来说，跌不是风险，涨不回来才是风险。** 而宽基指数长期向上，时间就是它的朋友。

### 但有些情况你确实该"止损"

| 什么情况 | 为什么该卖 | 这不是"止损"，是 |
|---------|-----------|----------------|
| 基金经理换了 | 当初买就是因为这个经理 | 理由消失了 |
| 基金规模暴增到 200 亿+ | 经理的操作空间变小 | 产品变味了 |
| 持仓风格大变 | 你说买大盘蓝筹，它买小盘题材 | 挂羊头卖狗肉 |
| 你当初买它的理由是错的 | 你承认当时没搞懂 | 纠正错误 |
| 你急需用钱 | 被迫的，没办法 | 流动性需要 |

> **因为"当初买它的理由不再成立"而卖 = 理性。因为"跌太多了害怕"而卖 = 情绪。**

## 给新手最实用的仓位建议

### 第一步：先建安全垫

```
在开始投资之前：
  1. 还清高息贷款（信用卡、网贷）
  2. 存够 6 个月生活费（放货币基金里）
  3. 买好保险（医疗险+重疾险）

→ 没有安全垫之前，先别急着投资。
```

### 第二步：从最小的仓位开始

```
不要一上来就投入全部积蓄。

第一个月：只投 1000 元
→ 体验一下波动是什么感觉
→ 看看你看到亏损时的真实反应

第二个月：如果心态 OK，再加到 2000
→ 慢慢加，不是一下子冲到目标仓位

→ 让你不舒服的不是"理论上的风险"，是"亲眼看到账户变红"。
```

### 第三步：用定投逐步建仓

```
不要一次全投进去。

目标仓位：10 万元投股票基金
建仓方式：分 10 个月，每月投 1 万

好处：
  → 分散了时点风险
  → 如果市场后来跌了，你在更低的位置继续买
  → 如果市场一路上涨，你也享受了部分涨幅
```

### 第四步：定期复盘，但不要太频繁

```
每季度看一次账户就够了：

问自己三个问题：
  1. 现在股票/债券的比例还符合我的目标吗？
  2. 我买的基金，买它的理由还在吗？
  3. 我的生活发生了需要用这笔钱的变化吗？

其他时间——别看。看了也没用，只会增加操作冲动。
```

---

## 这篇你该带走的三句话

1. **永远留一部分现金**——那不是"浪费"，是给你的心态买的保险
2. **宽基指数不需要止损**——时间就是它的朋友
3. **少看账户，多做功课**——看得越多，错得越多
