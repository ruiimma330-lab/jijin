# 开户实战指南

> 从来没买过基金？这篇手把手带你从零开始，5 分钟搞定开户。

## 一、在哪开户：三大渠道对比

买基金的"入口"有三种。对新手来说，**支付宝就够用了**，但你也可以了解全部选项再做决定。

| | 支付宝/蚂蚁财富 | 天天基金 | 券商APP |
|------|------|------|------|
| **全名** | 支付宝 → 理财 → 基金 | 天天基金网（东方财富旗下） | 华泰/中信/招商等 |
| **申购费率** | 1折（约0.15%） | 1折（约0.15%） | 1~1.5折 |
| **基金数量** | 全市场 | 全市场，最全 | 场外+场内ETF |
| **最低买入** | 1元 | 10元 | 场外1元，场内1手起 |
| **操作难度** | ★☆☆☆☆ 最简单 | ★★☆☆☆ 数据多稍复杂 | ★★★☆☆ 需要股票账户 |
| **适合谁** | **新手首选** | 想深入分析数据的人 | 也想买卖股票/ETF的人 |
| **开户门槛** | 有支付宝就行 | 手机号注册 | 需身份证+银行卡+视频认证 |

> **一句话建议**：新手用支付宝，等搞懂了再考虑开天天基金或券商账户。三个都开也行，费率差不多。

### 场内 vs 场外：一个买菜类比

- **场外基金**（支付宝/天天基金）：就像去超市买菜，明码标价，按当天收盘价成交。每天只有一个价格（净值），你下单后要等收盘才知道买价。
- **场内ETF**（券商APP）：就像去菜市场买菜，价格实时波动，你可以讨价还价。开盘时间随时买卖，价格每秒都在变。

**新手从场外开始**。超市买菜更省心，不用盯盘。

## 二、开户前准备

### 你需要准备的东西

| 物品 | 说明 |
|------|------|
| **身份证** | 二代身份证，在有效期内 |
| **银行卡** | 储蓄卡（借记卡），不支持信用卡 |
| **手机号** | 本人实名认证的手机号 |
| **一点点钱** | 10 元就够，真的 |

### 你不需要的东西

- ❌ 不需要去银行柜台
- ❌ 不需要工作证明
- ❌ 不需要收入流水
- ❌ 不需要找"理财经理"
- ❌ **不需要开股票账户**（这点很重要！基金账户和股票账户是两回事）

## 三、支付宝开户（推荐新手，3 分钟）

这是最简单的路径，因为你的支付宝大概率已经实名认证过了。

### 第1步：打开支付宝

在手机上打开支付宝 APP。你应该已经有账号了。

### 第2步：进入基金页面

底部菜单栏 → 点**「理财」** → 点**「基金」**。

如果你找不到"理财"，可以在首页搜索框直接搜"基金"。

### 第3步：完成风险测评（首次必须）

进入基金页面后，系统会提示你做**风险测评**。这是一份问卷，大概 10 道题：

- 你的年龄？
- 你的年收入？
- 你能接受多大的亏损？
- 你打算投资多久？
- 你之前有过投资经验吗？

**如实填写即可**。不要为了"通过"而选激进的答案——风险测评是为了保护你，让你不买自己扛不住的产品。测评结果分 C1~C5 五个等级，决定了你能买哪些类型的基金。

> 测评大约需要 2 分钟。做完之后就可以买基金了。

### 第4步：绑定银行卡（如果还没绑）

如果你支付宝里已经绑了银行卡，这步跳过。没有的话：支付宝 → 我的 → 银行卡 → 添加银行卡。

### 第5步：你开户完成了

是的，就这些。**没有单独的"开基金账户"按钮**。在支付宝完成风险测评后，你的基金账户就自动开通了。接下来就可以直接搜索基金、点买入。

### 完整流程回顾

```
打开支付宝 → 理财 → 基金 → 做风险测评(2分钟) → 完成 ✅
```

> 全程不需要离开支付宝，不需要额外下载 APP，不需要额外注册。

## 四、天天基金开户（数据党首选，5 分钟）

天天基金是东方财富旗下的专业基金平台，基金数据比支付宝更全，适合喜欢自己分析数据的人。

### 第1步：下载 APP

在手机应用商店搜**「天天基金」**，下载安装。

认准开发者是**「东方财富信息股份有限公司」**，别下到山寨版。

### 第2步：注册账号

打开 APP → 点**「我的」** → 点**「登录/注册」** → 输入手机号 → 获取验证码 → 设置密码。

### 第3步：绑定银行卡

登录后 → 「我的」→ 「银行卡」→ 添加银行卡（用于买基金和赎回）。

支持主流银行：工商、建设、招商、农业、中国银行等。

### 第4步：完成风险测评

首次进入基金相关功能，系统会提示做风险测评。和支付宝一样，如实填写。

### 第5步：完成

做完测评，开户就完成了。可以开始搜索和购买基金。

### 天天基金 vs 支付宝 选哪个？

| 场景 | 推荐 |
|------|------|
| 我就想简单定投，不想折腾 | 支付宝 |
| 我想看详细的基金数据、排名、对比 | 天天基金 |
| 我都想要 | 两个都开，费率一样 |

## 五、券商APP开户（进阶，10 分钟）

如果你未来可能想买股票或场内 ETF，可以开一个券商账户。如果只买基金，**不需要开**。

### 主流券商

华泰证券（涨乐财富通）、中信证券、招商证券、国泰君安等。

### 开户流程

1. 下载券商 APP → 手机号注册
2. 上传身份证正反面照片
3. 视频认证（对着摄像头念一段话，确认是本人）
4. 绑定银行卡（三方存管）
5. 做风险测评
6. 等待审核（通常几分钟到 1 个工作日）

> 券商开户比基金账户复杂，需要视频认证。对新手来说，先走完支付宝路径，需要时再来开券商账户。

## 六、常见问题

### Q1: 需要多少钱才能开始？

**10 元就够了。** 支付宝和天天基金上的大部分基金，最低买入 1~10 元。你不必一开始就投大钱——先买 10 元体验一下，熟悉了再增加。

### Q2: 安全吗？钱会不会没了？

两道安全保障：

1. **钱在托管银行，不在基金公司**。就算支付宝倒闭、基金公司倒闭，你的钱也在银行里。基金行业运行 25 年，从未出现过卷款跑路事件。
2. **基金公司受证监会严格监管**，必须定期披露持仓和净值，信息透明。

**不安全的不是平台，而是你自己的投资决策**——买了不适合的产品、追涨杀跌、频繁操作。

### Q3: 未成年人能开户吗？

**不能直接开户。** 法律规定 18 岁以下不能独立开基金账户。

但可以用**监护人的账户**来买——比如父母用自己的支付宝买基金，钱算父母的，但可以让孩子来了解投资。

### Q4: 开户要收费吗？

**完全免费。** 开基金账户不收任何费用。买基金时才收申购费（通常 0.1%~0.15%），但比去银行柜台买（1.2%~1.5%）便宜得多。

### Q5: 我没有支付宝，也没有银行卡，怎么办？

先去银行办一张储蓄卡（16 岁以上可自己办），然后注册支付宝并绑定银行卡。这是进入投资世界的两个基本工具。

### Q6: 我已经开了支付宝基金账户，还能在天天基金买吗？

**可以。** 不同平台的基金账户是独立的，互不影响。你在支付宝买的基金只存在于支付宝，在天天基金买的基金只存在于天天基金。如果你想知道自己一共持有多少基金，需要两边都查看。
