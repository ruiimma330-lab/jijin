# 定投为什么适合小白

## 什么是定投

> **定投** = 在固定的时间、投入固定的金额、买入同一只基金。

比如：每月 15 号，自动买入 1000 元沪深 300 指数基金。

## 定投解决了小白的三个难题

### 难题1：不知道什么时候买

没有人能精准预测市场底部。但定投**在不同时间点分批买入**，自动实现了"跌时多买，涨时少买"。

```
基金净值： 1.00 → 0.80 → 0.60 → 0.80 → 1.00
每次定投 1000 元：

净值 1.00 → 买 1000 份
净值 0.80 → 买 1250 份  (跌了，同样的钱买更多)
净值 0.60 → 买 1667 份  (再跌，买更多)
净值 0.80 → 买 1250 份
净值 1.00 → 买 1000 份

总投入：5000 元
总份额：6167 份
平均成本：0.81 元/份

回到 1.00 时：资产 = 6167 元 → 赚了 23.3%

对比一次性投入 5000 元 @1.00 → 回到 1.00 一分没赚
```

**定投在波动中创造了收益。** 这是定投最核心的魔法。

### 难题2：管不住手

看到大跌就想卖，看到大涨就想追——这是人性，几乎所有人都会犯。

定投是**自动执行**的纪律工具：设定好规则，不看盘，不纠结。

### 难题3：资金有限

一次性拿不出 10 万元，但每月拿 1000 元没问题。定投把大额投资变成了**可积累的小额储蓄**。

## 定投适合什么基金？

| 基金类型 | 适合定投吗 | 原因 |
|---------|----------|------|
| 货币基金 | ❌ | 波动太小，没有"摊低成本"的效果 |
| 债券基金 | ❌ | 波动不够大，效果不显著 |
| 指数基金(宽基) | ⭐⭐⭐ | **最推荐**，不会死、长期向上、费率低 |
| 主动股票基金 | ⭐⭐ | 要选对经理，否则可能越定投越亏 |
| 行业/主题基金 | ⭐ | 波动大效果好，但行业可能长期不涨 |

> **定投的黄金搭档是宽基指数基金**：沪深 300、中证 500、创业板指。

## 定投的三种策略

### 1. 普通定投（推荐新手）
每月/每周固定金额，不管涨跌都买。

```
优点：最简单，完全不用动脑
缺点：市场很贵的时候也在买
```

### 2. 智能定投（进阶）
根据估值或均线调整金额：便宜时多买，贵时少买。

```
参考信号：沪深 300 PE 分位数
PE < 30% 分位(低估) → 买 1.5 倍
PE 30-70% 分位(合理) → 买 1.0 倍
PE > 70% 分位(高估) → 买 0.5 倍或暂停
```

### 3. 目标定投（有特定用途）
设定目标金额，达到后止盈。

```
目标：攒够 50 万首付
每月定投 3000 元 + 年化 8%
约 10 年达成目标
```

## 定投最常踩的坑

| 坑 | 为什么危险 | 怎么避免 |
|---|----------|---------|
| **大跌就停** | 定投的收益就来自在低位多买，停了就白费 | 越跌越要坚持，甚至加码 |
| **涨了就想卖** | 复利需要时间，频繁止盈 = 放弃长期收益 | 设定明确的止盈规则，到了才卖 |
| **投得太杂** | 买十几只基金也分散不了系统性风险 | 2-3 只宽基指数足够 |
| **不看费率** | 1.5% 和 0.15% 的管理费，30 年差几十万 | 选低成本指数基金 |
| **钱不是闲钱** | 急需用钱时可能正好在亏损期 | 只投 3 年内用不到的钱 |

## 一个真实的定投计划模板

```
【目标】30 年后退休补充养老金，目标 200 万
【策略】每月 15 号定投
  - 1500 元 → 沪深 300 指数基金 (代码: 510300)
  - 1000 元 → 中证 500 指数基金 (代码: 510500)
  - 500 元  → 债券基金 (代码: 000015)
【止盈】不设止盈，退休时开始分批赎回
【调整】每年年底复盘一次，根据估值微调比例
【预期】年化 8%，30 年后约 380 万
```

## 这篇你该带走的三句话

1. 定投的核心不是"定"，是**低位多买**
2. 宽基指数基金 + 定投 = 新手的最佳起点
3. 越跌越买是反人性的，所以要自动化、少看盘
