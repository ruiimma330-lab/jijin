# 新手必读：基金投资学习路线图

## 这个项目是做什么的

帮你从零开始搞懂基金和A股投资——不只是看文章，而是**动手跑数据**、**看真实图表**、**做策略回测**。

## 你需要的准备

| 事项 | 说明 |
|------|------|
| Python 环境 | 3.12+，推荐用 `uv` 管理依赖 |
| 数学基础 | 初中数学就够，会用计算器 |
| 时间投入 | 每天 30 分钟，坚持 5-6 周 |
| 心态 | 投资是长期的事，不急不躁 |

## 学习路线图

```
第0步：先看白话入门
└── 跑 notebooks/00-白话入门.ipynb  → 用日常例子理解基金

第1周：认识基金
├── 读 knowledge/01-基金基础/   → 搞清楚基金到底是什么
├── 跑 notebooks/01-你的第一笔基金数据.ipynb  → 亲手抓数据
└── 成果：能分辨货币基金和股票基金，会查净值

第2周：理解投资
├── 读 knowledge/02-投资入门/   → 复利、风险、定投逻辑
├── 跑 notebooks/02-基金对比与筛选.ipynb
└── 成果：会用指标筛选基金，理解定投为什么有效

第3周：掌握分析方法
├── 读 knowledge/04-分析方法/   → 看懂指标、技术分析、综合评估
├── 跑 notebooks/03-定投策略回测.ipynb
└── 成果：能回测策略，会看估值分位

第4周：构建策略与风控
├── 读 knowledge/05-投资策略/   → 定投、网格、资产配置
├── 读 knowledge/06-风险管理/   → 认识风险、仓位管理
├── 跑 notebooks/04-构建投资组合.ipynb  → 整合所有技能！
└── 成果：能构建自己的投资组合，知道怎么控制风险

第5周：估值分析与实战
├── 读 knowledge/04-分析方法/01-看懂基金指标.md  → 复习指标含义
├── 跑 notebooks/05-市场估值分析.ipynb  → PE分位、股债性价比、市场打分
├── 跑 python scripts/analysis/performance.py --code <你的基金>  → 业绩归因
└── 成果：能判断市场位置，制定个性化的买入/卖出计划
```

## 工具使用指南

### 知识库 (knowledge/)
纯 Markdown 文档，按目录顺序阅读即可。每篇末尾有「下一步」链接。

### 脚本 (scripts/)
Python 命令行工具，可以直接运行：
```bash
# 数据获取
python scripts/data/fetch_fund.py        # 基金排行/净值
python scripts/data/fetch_index.py       # 指数估值/走势
python scripts/data/fetch_stock.py       # 个股行情/财务数据
python scripts/data/fetch_stock.py --market  # A股市场总览

# 分析工具
python scripts/analysis/fund_scanner.py  # 多维度基金筛选
python scripts/analysis/risk.py --code 110020   # 风险评估报告
python scripts/analysis/correlation.py --codes 110020,001632  # 相关性分析
python scripts/analysis/performance.py --code 110020  # 业绩归因

# 策略工具
python scripts/strategy/backtest.py --code 110020  # 策略回测对比
python scripts/strategy/signals.py --code 110020   # 市场择时信号
python scripts/strategy/portfolio.py --codes 110020,001632  # 组合优化
```

### Notebook (notebooks/)
交互式教程，逐格运行看结果。启动方式：
```bash
jupyter notebook notebooks/
```

## 关键概念速查

| 概念 | 一句话解释 |
|------|-----------|
| 基金 | 把钱交给专业经理帮你投资 |
| 净值 | 每份基金值多少钱 |
| 定投 | 固定时间投固定金额 |
| 复利 | 利滚利，时间越长越猛 |
| PE | 市盈率，多少年回本 |
| 回撤 | 从最高点跌了多少 |
| 夏普比率 | 每承担1单位风险赚了多少 |

## ⚠️ 重要提醒

1. 本项目**不构成投资建议**，所有数据和分析仅供参考
2. 历史业绩不代表未来表现
3. 先学习再投资，亏钱的往往是急于求成的
4. 定投 + 指数基金是小白的黄金起点
