# 定投策略详解

## 一句话理解

> 定投的精髓不是"定时定额"，而是**用纪律对抗人性**——在你害怕的时候逼你买，在你贪婪的时候不让你追。

## 回顾：为什么定投有效

在 `02-投资入门/02-定投为什么适合小白.md` 里讲过微笑曲线。这里用另一个角度再理解一次：

### 用买鸡蛋来理解

```
假设你每周买 10 块钱的鸡蛋：

普通买法：每周买一盒，不管价格
定投买法：每周花 10 块钱，便宜的周多买，贵的周少买

第1周：鸡蛋 10 元/斤  →  10元买 1 斤
第2周：鸡蛋 8 元/斤   →  10元买 1.25 斤  (降价了，同样钱多买25%)
第3周：鸡蛋 6 元/斤   →  10元买 1.67 斤  (继续降，多买67%)
第4周：鸡蛋 8 元/斤   →  10元买 1.25 斤
第5周：鸡蛋 10 元/斤  →  10元买 1 斤

总花费：50 元
总买到：6.17 斤
平均成本：50÷6.17 = 8.10 元/斤

第5周时鸡蛋回到 10 元：
你手里的值：6.17×10 = 61.70 元
赚了：11.70 元（+23.4%）

如果一次性在第1周买 50 元：
买 5 斤，到第5周值 50 元，一分没赚。
```

**定投让你"自动地"在便宜时多买。这是它唯一的魔法，但足够强大。**

## 三种定投策略对比

### 1. 普通定投（推荐新手）

```
怎么做：每月/每周固定金额，到了就买，不看盘
优点：  零决策成本，100% 执行纪律
缺点：  市场很贵时也在买
适合：  所有人（尤其是刚开始的）
```

### 2. 智能定投（进阶）

```
怎么做：根据 PE 估值或均线调整金额

PE 分位 < 30%  →  买 1.5 倍  （低估，多囤）
PE 分位 30-70% →  买 1.0 倍  （正常，照常）
PE 分位 > 70%  →  买 0.5 倍  （高估，少买）

优点：  理论上收益更高
缺点：  需要纪律，判断"分位"有主观性
适合：  有一定经验后
```

### 3. 目标定投

```
怎么做：设定目标金额，达到后止盈，重新开始

例：目标是攒 20 万
每月投 3000，年化 8%，约 4.5 年达成
到了 20 万就赎回盈利部分，把本金继续投

优点：  有明确的终点和目标感
缺点：  可能卖早了，踏空后面的涨幅
适合：  有具体花钱目标的人（首付、学费）
```

## 定投频率：每周好还是每月好？

**长期来看，几乎没区别。** 选你觉得最舒服的。

```
每月一次：
  优点：简单，和工资日对齐
  缺点：可能恰好买在当月最高点

每周一次：
  优点：更平滑，分散更细
  缺点：操作次数多，短期感觉不到差别

双周一次：
  折中方案
```

> 核心不是频率，是**坚持**。一个月投一次坚持了 5 年的人，远胜过每周投但投了半年就停的人。

## 定投最容易犯的五个错误

### 错误一：跌了就停

```
"都跌了 15% 了，我先暂停看看..."
→ 等你觉得"安全"了再开始，通常已经在半山腰以上了
→ 定投最大的收益来源就是你在低位捡的便宜筹码
→ 停了 = 放弃了定投唯一的核心优势
```

**正确做法**：越跌越要坚持，甚至暂时加码。（当然前提是投的是宽基指数，不是可能死掉的个股）

### 错误二：涨了就卖

```
"已经赚了 12% 了！赶紧落袋为安..."
→ 然后基金又涨了 40%，你只赚了 12%
→ 频繁止盈 = 放弃了复利最重要的因素：时间
```

**正确做法**：设定明确的止盈规则（比如年化 25% 以上才考虑），而不是凭感觉卖。

### 错误三：越买越多只

```
第1年：买沪深 300，不错
第2年：再加个中证 500，分散一下
第3年：消费基金涨得好，来一只
第4年：医药是永远的赛道，来一只
第5年：新能源是未来，来一只
...
→ 手里有 12 只基金，以为自己分散了
→ 一看持仓，全是 A 股，大环境不好一样跌
```

**正确做法**：2-4 只足够分散。宽基指数 + 债券 + 海外（如有），真正的分散是跨资产、跨市场的。

### 错误四：不停切换

```
"我买的这个沪深 300 管理费 0.2%，那个只要 0.15%！换！"
"这只最近3个月没那只涨得好，换！"
→ 每次切换都可能买在高点、卖在低点
→ 频繁交易的摩擦成本累积起来非常可怕
```

**正确做法**：买之前花 1 个月研究。买之后至少拿 1 年再评估。

### 错误五：不是闲钱

```
"每月投 5000，但下个月要交房租 10000..."
→ 急需用钱时，正好赶上市场大跌
→ 被迫在低点割肉，之前的定投白投了
```

**正确做法**：只投 3 年内绝对用不到的钱。留够 6 个月生活费在活期/货币基金里。

## 定投的止盈策略

什么时候该卖？三种思路：

### 1. 目标收益法
```
达到目标收益率（如年化 15%）→ 赎回盈利部分 → 本金继续定投
```

### 2. 估值法
```
PE 分位 > 80% → 赎回 50%
PE 分位 > 90% → 赎回剩余 50%
等 PE 分位回落到 50% 以下重新开始
```

### 3. 永不卖出（退休型）
```
一直定投到退休，然后每月赎回当月生活费
让复利跑满全程
```

---

## 这篇你该带走的三句话

1. 定投的唯一魔法是**自动化地低位多买**——不要主动破坏它
2. **坚持 > 频率 > 金额**——停了就什么都没有了
3. 用闲钱、不止损、不乱换——三条纪律保住你的收益
