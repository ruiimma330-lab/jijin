# 网格交易与波段操作

## 一句话理解

> 网格交易就像在菜市场摆摊——有人出高价你就卖给他，有人抛售你就低价收，赚的就是这个差价。

## 网格交易的核心逻辑

### 用水果摊理解

```
你把摊位设在菜市场，苹果的"正常价"是 10 元/斤。

你设置的网：
  12 元 → 卖 1 斤  （贵了，卖）
  10 元 → 不买不卖  （正常价，观望）
  8 元  → 买 1 斤  （便宜了，进货）

如果苹果从 10 → 8 → 12 → 10：
  在 8 元买了 1 斤（成本 8 元）
  在 12 元卖了 1 斤（收入 12 元）
  → 净赚 4 元（50% 收益！）
  
而苹果价格转了一圈回到 10 元，但你已经赚到钱了。
```

**这就是网格交易的精华：不预测方向，只在波动中赚差价。**

---

## 网格交易 vs 定投

| | 定投 | 网格交易 |
|---|------|---------|
| 操作方式 | 只看时间，到了就买 | 看价格，跌到位才买 |
| 卖出 | 一般不卖（长期持有） | 涨到位就卖 |
| 适合市场 | 长期向上的市场 | 横盘震荡的市场 |
| 资金利用 | 每次都全额投入 | 部分资金等待机会 |
| 复杂度 | 极低 | 需要设置和跟踪 |
| 适合谁 | 所有人 | 有一定经验后 |

> **定投是主食，网格是配菜。** 新手先吃好主食，再考虑加配菜。

## 如何设置一个网格

### 第一步：选标的
必须是**不会死、长期在区间内波动的品种**。

```
✅ 适合网格的：
  - 宽基指数 ETF（沪深 300、中证 500）
  - 债券基金
  - 黄金 ETF

❌ 不适合网格的：
  - 个股（可能退市）
  - 一直涨的标的（你会早早卖光）
  - 一直跌的标的（你会满手亏损）
```

### 第二步：定价中枢
网格需要一个"基准价"——你觉得差不多合理的价格。

```
怎么找中枢：
  - 看 MA120（半年线）的位置
  - 看 PE 中位数对应的点位
  - 看你最初买入的价格

例：沪深 300 在 4000 点 → 以此为中枢
```

### 第三步：设网格间距

| 类型 | 网格间距 | 特点 |
|------|---------|------|
| 宽基指数 | 3-5% | 波动适中 |
| 行业 ETF | 5-8% | 波动较大 |
| 债券基金 | 1-2% | 波动较小 |

### 第四步：分配资金

```
总资金：10 万元

如果设 5% 间距，上下各 4 格：
  每格金额：10万 ÷ 8 = 1.25 万
  基准价买入 50% = 5 万
  剩余 5 万分 4 份，等下跌时买入

价格上涨 5%  →  卖 1.25 万
价格上涨 10% →  卖 1.25 万
...
价格下跌 5%  →  买 1.25 万
价格下跌 10% →  买 1.25 万
```

## 网格交易的实操举例

```
沪深 300 当前 4000 点
网格间距：5%
每格资金：2000 元
初始买入：1 万元（5 份）

网格表：
  4400 → 卖 2000 元
  4200 → 卖 2000 元
  4000 → 基准（初始买入）
  3800 → 买 2000 元
  3600 → 买 2000 元

市场怎么走你都有应对：
  涨到 4200 → 自动卖一份（落袋为安）
  跌到 3800 → 自动买一份（低位进货）
  来回震荡 → 反复赚差价
```

## 网格交易的三个致命陷阱

### 陷阱一：单边上涨——你早早卖光了

```
你在 4000 点开始网格：
  4200 卖一份 ✓
  4400 卖一份 ✓
  4600 没货了 → 踏空 600 点的涨幅

→ 牛市里网格会踏空大部分收益
→ 所以网格更适合"卫星仓"，不是主力策略
```

### 陷阱二：单边下跌——子弹打光了还在跌

```
你在 4000 点开始网格：
  3800 买一份 ✓
  3600 买一份 ✓
  3400 买一份 ✓
  3200 没钱了 → 继续跌到 2800

→ 熊市里网格会让你抄底抄在半山腰
→ 补仓要有底线，不能无限补
```

### 陷阱三：摩擦成本

```
每次买卖一进一出：申购费 + 赎回费 ≈ 0.5-1.5%

如果一年交易 10 次，光手续费就花了 5-15%
→ 网格太小（如 2%），手续费把利润全吃光了
→ 网格越大，手续费影响越小，但成交机会也越少
```

## 你需要网格交易吗？

### 适合的情况
- 你已经有定投主体仓位了
- 有一笔额外的闲钱想赚点波动收益
- 能接受程序化执行（手动网格太累）
- 市场在横盘震荡期

### 不适合的情况
- 你是小白，连定投都没坚持下来
- 钱不多（< 5 万，网格分几份就没多少了）
- 没时间跟踪和调整
- 心态容易崩（补了 3 次还在跌就慌了）

---

## 这篇你该带走的三句话

1. 网格赚的是**波动的钱**，不预测方向
2. **定投为主，网格为辅**——别把配菜当主食
3. 单边涨你会踏空，单边跌你会补到没钱——网格不是万能
