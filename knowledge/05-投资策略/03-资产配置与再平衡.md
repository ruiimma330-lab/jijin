# 资产配置与再平衡

## 一句话理解

> 资产配置就像搭配一日三餐——光吃肉不行，光吃菜也不行。**关键是不同食物之间的搭配，而不是哪一道菜最好吃。**

## 为什么"全买最牛的基金"是错的

很多人会这样想：
```
"沪深 300 长期年化 10%，那我全买它不就好了？"
```

问题在于：
- 沪深 300 在 2008 年跌了 65%，2009-2014 年横盘 6 年没涨
- 如果你恰好在这段时间需要用钱，就完了

**资产配置的真正目的不是"赚最多"，而是"在任何年份都不至于崩溃"。**

### 用足球队理解配置

```
守门员（防守）：货币基金、短债    → 保命用的，别嫌收益低
后卫（稳健）：  中长期债券          → 让你在市场暴跌时还能睡得着
中场（均衡）：  宽基指数            → 主力部队，长期主攻
前锋（进攻）：  行业/主题基金        → 有机会就冲，但别放太多
```

**一支好的球队，每个位置都要有人。投资也一样。**

## 三类核心资产的性格

### 股票类（进攻型）

```
性格：脾气大，长期来看给的钱最多
年化：8-12%（长期）
波动：20-30%
适合放 3 年以上不动的钱
作用：让你变富
```

### 债券类（防守型）

```
性格：温顺，每年给一点固定零花钱
年化：3-5%
波动：2-5%
适合放 1-2 年可能用到的钱
作用：让你在市场暴跌时有钱补仓
```

### 现金类（守门员）

```
性格：躺平，但随时能拿出来
年化：1-2%
波动：0
适合随时可能用到的钱
作用：让你不至于被迫卖股票/基金
```

## 四种经典的配置方案

### 方案一：极简二八（推荐纯小白）

```
80%  债券基金
20%  沪深 300 指数

年化预期：4-6%
最大回撤：约 5-10%

→ 基本感受不到波动
→ 比存银行好，但也不用操心
→ 适合：完全不想管的人 | 钱 1-2 年内可能要用
```

### 方案二：六四配置（推荐大多数新手）

```
60%  股票类（沪深 300 + 中证 500）
40%  债券类（纯债基金）

年化预期：6-9%
最大回撤：约 15-20%

→ 兼顾了收益和稳健
→ 跌 20% 时，债券部分让你不至于恐慌性全卖
→ 适合：3-5 年不需要动这笔钱的人
```

### 方案三：核心-卫星（推荐有想法的新手）

```
70%  核心仓位（宽基指数，不动的压舱石）
  ├─ 40% 沪深 300
  └─ 30% 中证 500

20%  卫星仓位（行业/策略基金，看情况调整）
  └─ 医药/消费/科技（选 1-2 个你了解的行业）

10%  防守仓位（纯债）
  └─ 市场暴跌时的弹药库
```

### 方案四：全球化配置（进阶）

```
40%  A 股（沪深 300 + 中证 500）
20%  海外成熟市场（标普 500 / 纳斯达克）
10%  海外新兴市场
20%  国内债券
10%  黄金/商品

→ 跨市场分散，东方不亮西方亮
→ 适合：资金量大、追求更稳的人
```

## 再平衡——资产配置的"定期保养"

### 什么情况需要再平衡

```
你设定了 60% 股票 + 40% 债券

一年后：
  股票涨了 30%，债券涨了 3%
  股票变成了 65%，债券变成了 35%

→ 偏离了你的目标比例
→ 需要"再平衡"——卖掉一点涨得好的，补一点涨得差的
```

### 为什么需要再平衡

```
很多人不愿意："卖掉涨得好的？那不是傻吗？"

但再平衡的逻辑是：
  卖掉涨多了的 → 落袋为安
  补仓跌多了的 → 低位布局

  长期来看，再平衡 =
  自动执行了"高抛低吸"
  + 控制组合风险不失控
```

### 三种再平衡方法

| 方法 | 怎么做 | 适合谁 |
|------|--------|--------|
| 定时再平衡 | 每年固定一天调整（如每年 12 月） | 大多数人 |
| 阈值再平衡 | 偏离超过 5% 才调整 | 有纪律的人 |
| 用新钱再平衡 | 定投时多买比例低的 | 还在持续投入的人 |

> **推荐新手用"用新钱再平衡"**：不动老仓位，每月定投时多买那个比例偏低的，慢慢拉回来。

## 不同人生阶段的配置建议

```
【25 岁 — 刚工作】
  没啥钱但时间很多
  → 90% 股票 + 10% 债券
  → 你最大的资产是未来的工资，亏了还能挣

【35 岁 — 成家立业】
  有积蓄但责任也大
  → 70% 股票 + 30% 债券
  → 开始降低波动，保护已有成果

【50 岁 — 孩子上大学 / 准备退休】
  资金量大了
  → 50% 股票 + 50% 债券
  → 保本第一，收益第二

【65 岁 — 退休了】
  没有工资收入
  → 30% 股票 + 60% 债券 + 10% 现金
  → 主要是产生稳定的现金流
```

> **年轻时最大的优势不是本金，是时间。** 越年轻，越应该多配股票类资产。

---

## 这篇你该带走的三句话

1. **不把鸡蛋放一个篮子里**——但篮子之间要真的不一样
2. **再平衡 = 自动高抛低吸**——每年花 30 分钟调整一次
3. **随着年龄变大，慢慢从进攻转向防守**
