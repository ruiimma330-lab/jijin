# A股市场概述

## 什么是A股

> **A股** = 在中国内地注册、在沪深北交易所上市、用人民币交易的股票。

"股"在这里指的是**上市公司的一部分所有权**。你买了一手(100股)茅台 = 你拥有茅台公司的一小部分。

## A股市场的三层架构

```
┌──────────────────────────────────┐
│         中国证监会（监管）          │
├──────────────┬───────────────────┤
│   上海证券交易所  │  深圳证券交易所     │
│   主板 + 科创板  │  主板 + 创业板      │
├──────────────┴───────────────────┤
│   北京证券交易所 (服务中小企业)      │
└──────────────────────────────────┘
```

## 主要指数：市场的"体温计"

指数是一篮子股票的加权平均值，用来衡量市场整体表现。

### 四大核心指数

| 指数 | 代码 | 代表什么 | 特点 |
|------|------|---------|------|
| **沪深300** | 000300 | 沪深两市最大的300家公司 | 大盘蓝筹，A股"门面" |
| **中证500** | 000905 | 排名301-800的500家公司 | 中盘成长，弹性更大 |
| **创业板指** | 399006 | 创业板前100家 | 科技成长，波动极大 |
| **科创50** | 000688 | 科创板50家龙头 | 硬科技，上市门槛不同 |

### 新手应该关注哪个？

**沪深 300**。它是 A 股最有代表性的指数，也是新手定投的首选标的。

## A股的板块划分

### 按行业分（申万一级行业）

A股几千只股票分为 31 个一级行业：

| 大类 | 典型行业 | 特点 |
|------|---------|------|
| **周期** | 钢铁、煤炭、化工、有色 | 跟着经济周期走 |
| **消费** | 食品饮料、家电、医药 | 长期稳定、牛股集中营 |
| **科技** | 电子、计算机、通信 | 高成长但波动大 |
| **金融** | 银行、券商、保险 | 低估值、高分红 |
| **制造** | 新能源、机械、军工 | 制造业升级主角 |

### 按市值分（规模风格）

| 风格 | 代表指数 | 特点 |
|------|---------|------|
| 大盘价值 | 沪深300价值 | 低PE、高分红、稳 |
| 大盘成长 | 沪深300成长 | 高增长、高估值 |
| 小盘价值 | 中证500价值 | 小公司里的"便宜货" |
| 小盘成长 | 中证500成长 | 小公司里的"快马" |

**A股风格轮动明显**：有的年份大盘好，有的年份小盘好。新手不要押注一种风格，**均衡配置**。

## A股的交易规则（快速了解）

| 规则 | 内容 |
|------|------|
| 交易时间 | 9:30-11:30, 13:00-15:00 (工作日) |
| T+1 制度 | 今天买的明天才能卖 |
| 涨跌停板 | 主板 ±10%，创业板/科创板 ±20% |
| 最小单位 | 1 手 = 100 股 |
| 交易费用 | 佣金(万2.5左右) + 印花税(卖出 0.05%) |

**新手不用关心这些细节**——你买的是基金，基金经理替你交易。

## A股与基金的关系

作为新手，你**不直接买股票**，而是通过基金间接投资A股：

```
你 → 买基金 → 基金经理 → 挑选A股 → 构建股票组合
```

这样做的好处：
- 不用自己选股（会选股比你想象的难得多）
- 不用盯盘
- 分散风险（一只基金持有几十上百只股票）

## A股的历史特征

### 1. 牛短熊长
A股的特点是**急涨慢跌**。一轮大牛市可能只持续 1-2 年，但熊市可以持续 3-5 年。

### 2. 高波动
沪深 300 年波动率约 20-25%，远高于美股（~15%）。**定投正是在这种高波动中发挥作用。**

### 3. 政策影响大
A股被称为"政策市"，货币政策和监管态度对市场影响巨大。**但不要试图预测政策——坚持定投就好。**

### 4. 散户主导
A股散户占比高，导致情绪化交易多、追涨杀跌多。这反而给理性的长期投资者创造了超额收益空间。

## 这篇你该带走的三句话

1. A 股 = 中国内地股票市场，沪深 300 是你最需要认识的指数
2. 新手不直接买股票，而是通过基金间接投资 A 股
3. A 股波动大，正适合定投策略发挥"低位多买"的优势
