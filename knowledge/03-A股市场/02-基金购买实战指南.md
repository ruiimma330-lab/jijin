# 基金购买实战指南

> 学了半天理论，到底在哪买？怎么买？这篇给你答案。

## 在哪买基金

### 三大渠道对比

| 渠道 | 费率 | 基金数量 | 适合谁 |
|------|------|---------|--------|
| **支付宝/蚂蚁财富** | 1折（0.15%左右） | 全市场 | 新手首选，操作最简单 |
| **天天基金** | 1折 | 全市场 | 数据最全，适合分析 |
| **券商APP**（华泰/中信等） | 1-1.5折 | 场内+场外 | 也想买股票的人 |

> **新手推荐**：支付宝搜"基金"或下载"天天基金"APP。两个都开也行，费率一样。

### 场内 vs 场外

| | 场外基金 | 场内ETF |
|------|---------|------|
| **在哪买** | 支付宝/天天基金/银行 | 券商APP（需要股票账户） |
| **交易方式** | 按日终净值申购赎回 | 盘中实时买卖 |
| **起点** | 1元起 | 1手（100份）起 |
| **适合** | 定投、长期持有 | 波段操作、大额配置 |

**新手从场外开始**，支付宝就能搞定一切。等你搞懂了再考虑开股票账户买ETF。

## 开户 5 分钟搞定

### 支付宝买基金（最简单）

1. 打开支付宝 → 底部点"理财" → 点"基金"
2. 完成风险测评（5分钟，如实填写）
3. 就可以买了

### 天天基金（数据党首选）

1. 下载"天天基金"APP
2. 手机号注册 → 绑定银行卡
3. 完成风险测评
4. 开始使用

> 不需要开股票账户。支付宝/天天基金用的是**基金账户**，和股票账户是两回事。

## 你的第一笔投资

### 第一步：选一只指数基金

新手第一只基金，推荐**沪深300指数基金**。

理由：
- 跟踪A股最大的300家公司，不会暴雷
- 费率极低（管理费0.15-0.5%）
- 长期向上（GDP在增长，300家龙头就跟着涨）

**具体选哪只？** 搜"沪深300"，找这几个条件：

| 条件 | 为什么 |
|------|--------|
| 规模 > 10亿 | 太小可能清盘 |
| 成立 > 3年 | 有历史可以看 |
| 跟踪误差 < 2% | 说明紧跟指数 |
| 费率低 | 省到就是赚到 |

> 用本项目工具筛选：`python scripts/analysis/fund_scanner.py --type index --keyword 沪深300`

### 第二步：先买 100 元试试

别一上来就投大钱。先买 100 元感受一下：

1. 搜到基金 → 点"买入"
2. 输入金额 100 元
3. 确认支付
4. **然后忘了它**

等一周后再看，体验一下"持有基金是什么感觉"。

### 第三步：设置定投

体验 OK 之后，设置定投：

1. 基金页面 → 点"定投"
2. 金额：月收入的 10-20%（比如月入 1 万，每周投 250-500）
3. 周期：**每周四**（历史数据显示周四下跌概率略高）
4. 扣款方式：银行卡自动扣

> 定投的核心是"不看不改不间断"。市场跌了？你同样的钱买到了更多份额，那是好事。

## 买基金的常见坑

### 1. 不看费率

同样跟踪沪深300，有的基金收 0.15%，有的收 1.2%。**每年差 1%**，持有 10 年差出几十个百分点。

### 2. 追涨

看到某只基金最近涨了 50%，冲进去 → 大概率买在山顶。

**做相反的事**：看到基金跌了、身边人都在骂 → 那是买的时候。

### 3. 频繁买卖

基金申购赎回有手续费（持有 < 7 天赎回费高达 1.5%）。频繁操作 = 给平台送钱。

**买完就放着**，至少持有半年以上。

### 4. 买太多只

新手一上来买十几只基金，号称"分散风险" → 其实只是买了个大盘。

**3-5 只足够**。沪深300 + 中证500 + 债券基金，三只就覆盖了大部分需求。

### 5. 看排行榜买

天天基金首页推荐的"近期热门" → 往往是涨完之后的。**你买进去的时候，行情可能已经结束了。**

## 买入之后做什么

| 时间 | 动作 |
|------|------|
| **每天** | 什么都不做。不要看账户。 |
| **每周** | 定投自动扣款，瞄一眼就行 |
| **每月** | 记一笔账：投入多少、市值多少 |
| **每季度** | 跑一次 `fund_scanner.py` 看看基金有没有大的变化 |
| **每年** | 用 `scripts/strategy/portfolio.py` 做一次再平衡 |

## 什么时候卖

新手最容易犯的错是"赚了一点就想卖"。这里给一个简单的规则：

- **基金本身没问题（规模稳定、跟踪误差小）** → 一直持有，只买不卖
- **需要用钱** → 卖一部分
- **市场极度亢奋**（身边不投资的人都在聊股票）→ 可以卖一部分
- **达到你的目标**（比如攒够首付）→ 卖

> "最好的卖出时机是永远不卖。" —— 很多老基民的感悟

## 本篇要点

1. 支付宝/天天基金就能买，不需要股票账户
2. 第一只基金选沪深300指数基金，先买100元试试
3. 定投是最好的策略：设好就不管，跌了是赚
4. 3-5只基金足够，别追涨，别看排行榜买
5. 买入后忘了它，半年看一次就行
