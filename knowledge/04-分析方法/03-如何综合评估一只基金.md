# 如何综合评估一只基金

## 一句话理解

> 选基金就像选餐厅——不能只看门口排队人数（热度），还要看食材品质（底层资产）、厨师水平（经理）、价格（费率）和服务体验（持有感受）。

## 为什么人总选错基金

### 最常见的错误：看排行榜买基金

```
去年涨了 80%  →  冲进去  →  今年跌了 30%
去年涨了 30%  →  不买     →  今年涨了 15%
```

**追涨杀跌是人的本能，但却是投资里最昂贵的错误。**

基金的收益往往"均值回归"——涨多了要歇，跌多了会涨。去年的冠军，经常是明年的倒数。这不是偶然，是因为：
1. 某一种风格不会永远好
2. 规模暴涨后基金经理操作变难
3. 运气成分被当成实力

---

## 评估基金的五步法

### 第一步：看底层资产（它到底买了什么）

别只看基金名字，打开"基金持仓"看看：

```
基金名字：XX 新兴成长混合
你以为的：投资新兴产业的成长型公司
实际上：
  贵州茅台  8.2%
  五粮液    6.5%
  宁德时代  5.8%
  ...
→ 其实就是白酒+新能源，和"新兴成长"关系不大
```

**关键检查**：
- 前十大持仓集中度 > 60%？→ 集中押注，波动会很大
- 持仓股票你认识吗？→ 如果不认识，说明你对该基金投了什么一无所知
- 投资方向和你以为的一致吗？→ 名不副实的基金太多了

---

### 第二步：看人（基金经理靠不靠谱）

| 看什么 | 怎么看 | 红线 |
|--------|--------|------|
| 从业年限 | 管理这只基金多久 | < 3 年，牛市经验可能不足 |
| 历史业绩 | 经历过牛熊吗 | 只经历过牛市的"大神"要警惕 |
| 管理规模 | 管了多少钱 | 管太多(>200亿)操作灵活性下降 |
| 跳槽频率 | 换过几家基金公司 | 频繁跳槽的稳定性存疑 |
| 投资理念 | 看季报里的"基金经理说" | 说不清楚自己怎么赚钱的，靠蒙 |

> **一个基金经理至少要看 3 年以上，最好跨越一轮牛熊。**

---

### 第三步：看数字（四个核心指标）

用 `04-分析方法/01-看懂基金指标.md` 里学到的方法：

```
              你的基金     同类平均     判断
年化收益       12.3%       10.5%       👍 高于平均
年化波动率     22.1%       18.0%       ⚠️ 比同类颠簸
最大回撤       -28.5%     -20.3%      ⚠️ 跌得够狠
夏普比率        0.56       0.58        ➖ 差不多
```

**口诀：收益比平均高、波动比平均低、回撤可控、夏普不差——这才是好基金。**

---

### 第四步：看价格（费率有多贵）

别小看费率。**每年多 1%，30 年可能差一倍：**

```
基金 A：管理费 0.15%/年（指数基金）
基金 B：管理费 1.5%/年（主动基金）

投入 10 万，年化 8%，30 年：

A: 10万 × (1.0785)^30 = 96.4 万
B: 10万 × (1.065)^30  = 66.1 万

差了 30 万！就因为每年多收了 1.35%
```

| 费用项 | 指数基金 | 主动基金 |
|--------|---------|---------|
| 管理费 | 0.15-0.5% | 1.0-1.5% |
| 托管费 | 0.05-0.1% | 0.15-0.25% |
| 申购费 | 0-0.1%(平台打折) | 0.1-0.15% |
| 赎回费 | <7天 1.5%, >1年免 | 同左 |
| **合计** | **约0.3-0.7%/年** | **约1.5-2.0%/年** |

---

### 第五步：看持有体验（你能不能拿得住）

这是最被忽视但最重要的一步。

```
基金 A：年化 12%，但最大回撤 35%
  → 你投入 10 万，中途变成 6.5 万，你卖了吗？

基金 B：年化 9%，最大回撤 15%
  → 你投入 10 万，中途最低 8.5 万，你可能根本没感觉
```

**最终投资者赚到的钱 ≠ 基金净值增长**

因为大多数人在高点冲进去，低点割肉出来。一只"拿得住"的基金比一只"涨得快但你拿不住"的基金，对你更有价值。

---

## 一份完整的基金评估清单

当你下一次考虑买一只基金时，对照这份清单：

```
□ 1. 持仓：前十大是什么？方向和你以为的一致吗？
□ 2. 经理：干了多久？经历过熊市吗？
□ 3. 收益：3年年化多少？和同类比如何？
□ 4. 回撤：最大跌过多少？你能承受吗？
□ 5. 夏普：性价比如何？
□ 6. 费率：一年吃你多少钱？
□ 7. 规模：基金规模 5-100 亿吗？
□ 8. 相关性：和你已有的基金有没有同涨同跌？
□ 9. 持有：你能想象自己拿 3 年不卖吗？
```

## 新手选基金的实用建议

### 最推荐的起点组合（3 只就够了）

```
50%  →  沪深 300 指数基金  →  A 股 300 家最大公司
30%  →  中证 500 指数基金  →  中盘成长股
20%  →  纯债/短债基金      →  降低波动，稳稳的幸福
```

- 管理费加起来 < 0.5%/年
- 不会因为某只股票暴雷而大亏
- 长期历史年化 6-10%
- 不需要判断基金经理的水平

### 如果你想加入一只主动基金（不超过 20%）

只有满足以下才考虑：
- 基金经理干了 5 年以上
- 经历过 2018 年熊市和 2022 年熊市
- 规模没超过 100 亿
- 你能说清楚他靠什么赚钱
- 管理费 < 1.5%

---

## 这篇你该带走的三句话

1. **买之前先做"五步检查"**——持仓、经理、指标、费率、持有体验
2. **宽基指数是你最好的起点**——分散、便宜、省心
3. **拿得住比涨得快更重要**——选你能安稳持有 3 年的基金
