# 看懂基金的核心指标

## 一句话理解

> 看基金就像相亲——不能只看"收入"(收益率)，还得看"脾气"(波动)、"抗压能力"(回撤)和"性价比"(夏普)。

## 生活中的类比

想象你在挑两个司机，送你走一段山路：

| | 司机A | 司机B |
|---|-------|-------|
| 到达时间 | 3小时 | 2.5小时 |
| 路上急刹车次数 | 3次 | 20次 |
| 最快速度 | 80码 | 150码 |
| 是否闯红灯 | 没有 | 闯了2个 |
| 你坐完的感受 | 稳稳当当，睡着到 | 心惊肉跳，一路抓紧扶手 |

**你会选谁？** 大概率选 A，虽然慢了 30 分钟，但踏实。

基金也一样——**高收益如果伴随着心惊肉跳的波动，不一定比稳稳的中等收益更好。**

## 指标一：年化收益率（Annual Return）

### 是什么
把基金的收益折算成"一年能赚多少%"的尺度。

### 举例子
```
基金过去 3 年从 1.00 涨到 1.30
总收益 = 30%
年化收益 ≈ 9.1%  (不是 30%÷3=10%，要考虑复利)

公式：(1.30)^(1/3) - 1 = 0.091 = 9.1%
```

### 看懂数字
| 年化收益 | 什么意思 |
|---------|---------|
| < 0% | 亏钱了 |
| 0-3% | 和存银行差不多 |
| 3-8% | 正常偏稳 |
| 8-15% | 不错 |
| 15-25% | 很好，但要看看风险 |
| > 25% | 很好，但大概率不可持续 |

### ⚠️ 常见陷阱
- **短期年化骗人**：某基金上个月涨了 5%，年化就是 `(1.05)^12 - 1 = 79.6%`。可能吗？不可持续。
- **幸存者偏差**：排行榜上看到的都是"活下来"的，清盘的你根本看不到。
- **过去不代表未来**：去年冠军经常是明年倒数。这叫"冠军魔咒"。

## 指标二：波动率（Volatility）

### 是什么
净值上蹿下跳的剧烈程度。**波动率越高，心脏需要的承受力越强。**

### 用温度来理解

```
体温恒定 37°C  →  低波动  →  货币基金
气温一天 20-30°C  →  中波动  →  债券基金
气温一天 5-35°C  →  高波动  →  股票基金
```

### 看懂数字（年化波动率）
| 波动率 | 什么意思 | 适合谁 |
|-------|---------|--------|
| < 5% | 稳如狗 | 1年内要用钱的人 |
| 5-15% | 正常心跳 | 能接受偶尔小亏 |
| 15-25% | 刺激 | 3年以上不用的闲钱 |
| > 25% | 过山车 | 铁石心肠 |

## 指标三：最大回撤（Max Drawdown）

### 是什么
**从最高点跌到最低点的最大跌幅。** 这是最重要的风险指标。

### 用买房理解

```
你买的房子：
买入价：300 万
最高到：400 万  →  开心
后来跌到：280 万  →  心痛

最大回撤 = (280-400) ÷ 400 = -30%
```

虽然现在可能又涨回来了，但那段时间你多煎熬？

### 回撤对你的实际影响

```
你有 10 万元，买了一只基金：

跌 10%  →  剩 9 万   →  涨 11% 回本
跌 20%  →  剩 8 万   →  涨 25% 回本
跌 30%  →  剩 7 万   →  涨 43% 回本  ← 开始痛苦
跌 50%  →  剩 5 万   →  涨 100% 回本 ← 很多人在这时割肉
```

**关键洞察：跌得越深，需要涨得越多才能回本。** 这不是对称的！

### 看懂数字
| 最大回撤 | 什么感觉 |
|---------|---------|
| < 10% | 稳稳的 |
| 10-20% | 有点心疼 |
| 20-30% | 晚上可能睡不好 |
| 30-50% | 很多人会割肉 |
| > 50% | 极度痛苦 |

> **诚实问自己**：你能接受账户亏 30% 还不卖吗？如果答案是"不能"，就选回撤小的基金。

## 指标四：夏普比率（Sharpe Ratio）

### 是什么
**每承担一单位风险，能换来多少超额收益。** 简单说就是"性价比"。

### 用买菜理解

```
菜市场 A：走 10 分钟，菜价便宜 5 块
菜市场 B：走 30 分钟，菜价便宜 6 块

A 的"夏普比率"更高——每走 1 分钟省 0.5 元。
B 每走 1 分钟只省 0.2 元，多走的路不划算。
```

基金的夏普比率 = (基金收益 - 无风险收益) ÷ 波动率

分子是"多赚的"，分母是"承受的颠簸"。

### 看懂数字
| 夏普比率 | 什么意思 |
|---------|---------|
| < 0 | 不如存银行 |
| 0-0.5 | 性价比不高 |
| 0.5-1.0 | 还行 |
| 1.0-2.0 | 不错 |
| > 2.0 | 非常好 |

## 指标五：胜率（Win Rate）

### 是什么
**有多少个交易日是上涨的。** 比如 60% 的胜率 = 10 天里有 6 天上涨。

### 但注意
胜率不是越高越好：
```
基金 A：胜率 55%，赚的日子平均 +0.3%，亏的日子平均 -0.1%  →  赚大亏小，👍
基金 B：胜率 80%，赚的日子平均 +0.05%，亏的日子平均 -2%  →  赚小亏大，👎
```

要看**盈亏比**——赚的时候平均赚多少，亏的时候平均亏多少。

## 怎么看一份基金的"体检报告"

以沪深 300 指数基金为例：

```
年化收益：     9.5%   →  不错
年化波动率：  18.2%   →  有点颠
最大回撤：    -22.3%  →  能承受吗？10万变7.8万
夏普比率：     0.52   →  还行，不算突出
胜率：        53%    →  和抛硬币差不多
```

**综合判断**：这是一只正常偏好的基金。长期拿着能赚钱，但中间肯定有让你想卖的时候。适合有耐心的人。

---

## 这篇你该带走的三句话

1. **看收益更要看回撤**——跌 50% 要涨 100% 才能回本
2. **夏普比率 = 性价比**——同样的收益，选波动小的
3. **历史数据只能参考**——过去的冠军可能是未来的倒数
