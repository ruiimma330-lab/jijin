# 技术指标入门

## 一句话理解

> 技术指标就像汽车仪表盘——不是用来预测未来的，而是告诉你**现在**发动机什么状态、速度快慢、油量多少。

## 重要警告：先看完这段再往下

技术指标有两大局限：

1. **滞后性**：所有指标都是基于**过去**的价格算出来的，等它发出信号，行情可能已经走了一半
2. **不是魔法**：没有任何指标能 100% 预测涨跌。如果有，发明它的人早就世界首富了

所以技术指标的正确用法是：**辅助判断，不是决策依据。**

---

## 一、均线（MA — Moving Average）

### 是什么
**过去 N 天价格的平均值，连成一条线。** 用来平滑掉短期波动，看清大方向。

### 生活类比

```
你的体重：
       周一 70kg  周二 71kg  周三 69kg  周四 70kg  周五 72kg

直接看 → 上蹿下跳，不知道趋势
算5日均线 → (70+71+69+70+72) ÷ 5 = 70.4kg
再算下一个5天 → ...连成一条平滑的线

⇒ 能看清楚"总体在变胖还是变瘦"的趋势
```

### 常用均线

| 均线 | 代表什么 | 怎么看 |
|------|---------|--------|
| MA5（周线） | 一周的趋势 | 太敏感，噪音多 |
| MA20（月线） | 一个月的趋势 | 短期方向 |
| MA60（季线） | 一个季度的趋势 | 中期方向 |
| MA120（半年线） | 半年的趋势 | 长期方向 |
| MA250（年线） | 一年的趋势 | 牛熊分界线 |

### 怎么用
```
价格 > MA20 → 短期偏强
价格 < MA20 → 短期偏弱

MA20 向上拐头 → 趋势可能变好
MA20 向下拐头 → 趋势可能变差
```

## 二、均线的"金叉"和"死叉"

### 是什么
两条均线交叉发出的信号。

```
金叉（Golden Cross）：
  MA20 从下往上穿过 MA60
  → 短期趋势正在超过长期趋势
  → 可能是上涨的开始

死叉（Death Cross）：
  MA20 从上往下穿过 MA60
  → 短期趋势正在跌破长期趋势
  → 可能是下跌的开始
```

### 生活类比

```
你每天跑步的速度：
- 短期(5天)平均配速：6分/公里
- 长期(20天)平均配速：7分/公里

短期超过长期 → 金叉 → 你最近越跑越快了！👍

短期低于长期 → 死叉 → 你最近变慢了 👎
```

### ⚠️ 注意
金叉死叉在震荡市会反复出现，让你频繁买卖、亏手续费。只在趋势明显的市场好用。

## 三、MACD — 均线的升级版

### 是什么
MACD 在均线的基础上更进一步，不仅看出方向，还能看出**动能在加强还是减弱**。

### 用电梯理解

```
DIF（快慢线差） = 你当前的上升速度
DEA（信号线）   = 你平均的上升速度
MACD 柱        = 当前速度和平均速度的差距

电梯上升时：
  MACD 柱变长 → 在加速上升 🚀
  MACD 柱变短 → 还在上升但减速了
  MACD 柱变负 → 开始下降了（DIF < DEA）
```

### 核心看三个东西
1. **DIF 和 DEA 的位置**：在零轴上方 = 偏强；下方 = 偏弱
2. **MACD 柱的方向**：柱子由绿变红且变长 = 上涨动能在增强
3. **背离**：价格创新低但 MACD 不创新低 → 可能跌不动了（底背离）

## 四、RSI — 相对强弱指标

### 是什么
**衡量最近涨得是不是太猛（超买）或跌得是不是太过（超卖）。** 数值在 0-100 之间。

### 用弹簧理解

```
把弹簧往下压 → 压得越深，反弹力越强
把弹簧往上拉 → 拉得越开，回缩力越强

RSI < 30 → 弹簧被压得很深 → 超卖 → 可能反弹
RSI > 70 → 弹簧被拉得很开 → 超买 → 可能回调
RSI 30-70 → 弹簧在正常范围
```

### 看懂数字
| RSI 值 | 什么状态 | 该干嘛 |
|--------|---------|--------|
| 0-20 | 极度超卖 | 别卖了，等反弹 |
| 20-30 | 超卖 | 关注 |
| 30-70 | 正常 | 该干嘛干嘛 |
| 70-80 | 超买 | 谨慎，别追了 |
| 80-100 | 极度超买 | 减仓或止盈 |

### ⚠️ 注意
在强牛市中，RSI 可以一直在 70 以上不回调。在熊市中，RSI 可以一直在 30 以下不反弹。不要机械使用。

## 五、布林带（Bollinger Bands）

### 是什么
**三条线组成的一条"通道"，价格大部分时间在这条通道里走。**

```
上轨 = 均线 + 2倍标准差  ← 天花板
中轨 = 均线             ← 中间线
下轨 = 均线 - 2倍标准差  ← 地板
```

### 用河流理解

```
河流两岸是上轨和下轨，水在中间流。

正常情况下：水流在两岸之间
冲上河岸 → 水太多了（极度强势）→ 但最终会回到河里
撞到河底 → 水太少了 → 但不会一直干涸
```

### 两个信号
1. **收口**（带宽变窄）：暴风雨前的宁静，可能要有大波动了
2. **张口**（带宽变宽）：波动开始加大，趋势可能明确

## 新手用技术指标的正确姿势

### ❌ 错误用法
```
"RSI 到 30 了！全仓买入！"
"金叉了！冲啊！"
"三个指标都说要涨，稳了！"
```

### ✅ 正确用法
```
"RSI 到 25 了，估值也不贵，我本来就在定投，不用慌。"
"金叉了，但 PE 还在高位，我先不动。"
"三个指标都偏乐观，市场情绪好像不错，但我还是按计划来。"
```

### 给新手的建议

1. **刚开始只看均线就够了**——方向比精确的买卖点重要一百倍
2. **不要在震荡市频繁用技术指标做决策**——会被反复打脸
3. **技术指标 + 估值 + 定投纪律**——三把锁比一把锁安全
4. **同一个指标不要看三条不同参数的**——那是假装在分析

---

## 这篇你该带走的三句话

1. 技术指标是仪表盘，不是水晶球——看状态，不预测未来
2. 新手从均线开始就够，一条线看懂方向
3. 任何指标单独使用都不可靠，要配合估值和纪律
