# 基金分类与风险等级

## 按投资对象分类（最重要）

基金按"买什么"分为五大类，风险从低到高：

```
风险低 ←————————————————————————→ 风险高
货币基金 < 债券基金 < 混合基金 < 股票基金 < 另类基金
```

### 1. 货币基金 — "余额宝就是它"

**投什么**：银行存款、短期债券、央行票据等极低风险资产  
**特点**：几乎不亏钱、随取随用、收益比银行活期高  
**年化收益**：1.5% ~ 2.5%  
**适合**：短期闲钱、应急资金  
**例子**：余额宝(天弘余额宝货币)、零钱通

### 2. 债券基金 — "稳健理财首选"

**投什么**：国债、企业债、金融债等固定收益资产  
**特点**：风险较低、收益比货基高、偶尔小幅回调  
**年化收益**：3% ~ 6%  
**适合**：1-3 年不用的钱、追求稳健  
**注意**：债基 ≠ 保本，2022 年底债市大跌很多债基亏了 2-3%

### 3. 混合基金 — "攻守兼备"

**投什么**：股票 + 债券，比例灵活调整  
**特点**：比纯股票稳、比纯债券猛  
**年化收益**：-10% ~ +30%（波动很大）  
**适合**：能接受波动的中长期资金

混合基金又分：
| 子类型 | 股票比例 | 风险 |
|--------|---------|------|
| 偏债混合 | < 30% | 中低 |
| 平衡混合 | 30-70% | 中 |
| 偏股混合 | > 70% | 中高 |

### 4. 股票基金 — "搏收益的主战场"

**投什么**：80% 以上买股票  
**特点**：涨跌剧烈、长期收益可能最高、短期可能大亏  
**年化收益**：-30% ~ +50%（波动极大）  
**适合**：3 年以上不用的钱、能承受腰斩的心理

### 5. 另类基金 — "小众选手"

包括：黄金 ETF、REITs(房地产信托)、商品期货基金等。新手先跳过。

## 按运作方式分类

### 开放式基金 vs 封闭式基金

| | 开放式 | 封闭式 |
|------|--------|--------|
| 随时买卖 | ✅ 可以 | ❌ 有固定期限 |
| 份额可变 | ✅ 可变 | ❌ 固定 |
| 交易场所 | 基金公司/代销平台 | 证券交易所 |
| 新手友好 | ✅ | ❌ 更复杂 |

你日常在支付宝上买的基本都是**开放式基金**。

### ETF（交易所交易基金）

ETF 是一种特殊的开放式基金，它**像股票一样在交易所买卖**：

| 普通基金 | ETF |
|---------|-----|
| 每天只有一个净值 | 交易时间价格实时变动 |
| 在基金平台申购赎回 | 在证券账户买卖 |
| 手续费较高(0.1-0.15%) | 手续费低(万分之几) |

**新手建议**：先从场外普通基金入手，熟悉之后再接触 ETF。

## 按投资风格分类

### 主动基金 vs 被动基金(指数基金)

| | 主动基金 | 被动基金(指数基金) |
|------|---------|-----------------|
| 目标 | 跑赢市场 | 跟踪市场 |
| 管理费 | 高 (1.2-1.5%) | 低 (0.15-0.5%) |
| 依赖 | 基金经理能力 | 指数规则 |
| 长期表现 | 多数跑输指数 | 接近指数 |

**巴菲特推荐普通人买指数基金**。成本低、不会踩雷、长期收益跑赢大多数主动基金。

## 风险等级速查

| 基金类型 | 风险等级 | 最大亏损参考 | 推荐持有期 |
|---------|---------|------------|-----------|
| 货币基金 | R1(低) | ~0% | 随时 |
| 短债基金 | R2(中低) | ~-1% | 3个月+ |
| 长债基金 | R2(中低) | ~-5% | 1年+ |
| 偏债混合 | R2(中低) | ~-5% | 1年+ |
| 平衡混合 | R3(中) | ~-15% | 2年+ |
| 偏股混合 | R3(中) | ~-25% | 3年+ |
| 股票基金 | R4(中高) | ~-40% | 3年+ |
| 指数基金 | R4(中高) | ~-40% | 3年+ |

## 新手选基三步法

```
第1步：确定这笔钱能放多久 → 决定了你能投什么类型
第2步：确定你能接受亏多少 → 决定了你的风险等级上限
第3步：在对应的类型里选最好的 → 后面章节会教你怎么选
```

## 这篇你该带走的三句话

1. 五大类型从低风险到高风险：货基 < 债基 < 混合 < 股票
2. 小白起步推荐：货币基金(日常闲钱) + 指数基金(长期投资)
3. 能放多久、能亏多少——这两个问题先回答清楚再投
